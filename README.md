# ECE2071_TeamJ02
This is a repository containing work for the course ECE2071, offered in Semester 1 at Monash University.

## Description
The purpose of this repository is to track the development of code for the ECE2071 Project B.
This repository contains versions of code made by each team member. This is managed through 7 branches:
- **main**- is where all final code is stored
- **Mani, Adam, Rohan, Ayra, Caitlin, Kiera**- branches for each team member, organised by team member's names

## Getting Started
### Dependencies
 Python- version 3.8 or later
- Operating system

**Tools:**
- Terminal or command prompt- to run the program
- Text editor- eg VS Code
- Installed Git

  ### Installing
#### Using GitHub:
1. Download the Project- by cloning the git repository using the GitHub url.
   ```bash
   git clone https://github.com/Kiera-Di-Sario/ECE2071_TeamJ02
   ```
